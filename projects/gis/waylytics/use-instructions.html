Getting Started

1. Creating Your First Trip

  • Navigate to "Trips" in the main menu
  • Click "Create New Trip" button
  • Enter trip name, dates, and description
  • Add locations with arrival/departure dates

2. Using Analytics

  • Access the Analytics dashboard
  • Set date ranges to filter data
  • Select specific trips for analysis
  • View metrics: total distance, days traveled, places visited

3. Custom Analysis

  • Use the Analysis dropdown for specific insights
  • View average stay duration
  • Check most visited places
  • Analyze longest trips
  • Review travel patterns

4. Data Management

  • All data is stored locally in your browser
  • Use export feature to backup your data
  • Import previous backups when needed
  • Clear data from settings if required

Tips for Better Analysis

  • Enter accurate dates for better timeline views
  • Add detailed location information for distance calculations
  • Use consistent naming for places to track visits
  • Regularly backup your travel data
