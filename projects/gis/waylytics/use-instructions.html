Getting Started with Waylytics

1. Home Page Overview

  • Visit the main page to see an interactive map displaying all your saved trips
  • Use the "Add New Entry" button on the right to start creating trips
  • Existing trips are visualized directly on the map

2. Managing Your Trips

  • View all trips in the Trip Management page
  • Create new trips using the "Create New Trip" button
  • Add places to existing trips via "Add New Place" button next to each trip
  • Edit or delete trips using the provided links

3. Adding Places to Trips

  • Use the search bar in the top right to find locations
  • Click "Draw a Marker" button in the top left map controls
  • Place your marker on the desired location
  • Fill in the place details in the form provided
  • Save to add the place to your trip

4. Data Analytics & Management

  • Access detailed trip analytics in the Data Analytics dashboard
  • Use Settings to manage your data:
  • Export your travel data as JSON or CSV for external analysis
  • Clear all locally stored data if needed

Pro Tips

  • Always return to the home page to verify your new entries on the main map
  • Regularly export your data to prevent loss of travel information
  • Use consistent location names for better tracking and analysis
  • Check the analytics dashboard to gain insights about your travel patterns
