How to Use LULC Mapping Sample Collector

1. Getting Started

Preparing Your Data

  • Ensure your GeoTIFF file is properly georeferenced
  • The GeoTIFF should be in a supported coordinate reference system (CRS)
  • Recommended: Use a multi-band image (e.g., RGB or multispectral)

2. Managing Classes

Default Classes

The tool comes with four default classes:

  • Water (Blue)
  • Dense Vegetation (Dark Green)
  • Agriculture (Light Green)
  • Urban Habitation (Grey)

Customizing Classes

To customize classes:

  1. Click the "Manage Classes" button
  2. Add new classes using the Add Class form
  3. Choose a name and color for your class
  4. Remove unwanted classes using the Remove button
  5. Click "Done" to save your changes

3. Collecting Samples

Loading Your Image

  1. Click "Choose GeoTIFF File" to upload your image
  2. The image will be displayed on the map
  3. The CRS information will be shown below the map

Adding Sample Points

  1. Select a class from the class buttons
  2. Click on the map to add sample points
  3. Points will be colored according to their class
  4. Click on any point to remove it

4. Exporting Data

Downloading Samples

  • Click the "Download CSV" button to export your samples
  • The CSV will contain point coordinates and class labels
  • Use the Clear All button to start over if needed
