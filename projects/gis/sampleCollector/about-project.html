About LULC Mapping Sample Collector

Project Overview

The LULC (Land Use Land Cover) Mapping Sample Collector is a web-based tool designed to facilitate the collection of training samples for land cover classification. It provides an intuitive interface for users to annotate satellite imagery and collect georeferenced points for different land cover classes.

Key Features

  • GeoTIFF Support: Load and visualize georeferenced satellite imagery
  • Custom Classes: Create and manage your own land cover classes with custom colors
  • Interactive Mapping: Click-to-collect point samples with precise coordinate tracking
  • CRS Awareness: Maintains coordinate reference system consistency
  • Data Export: Download collected samples in CSV format for further analysis

Technical Details

Built With

  • Leaflet.js for interactive mapping
  • GeoTIFF.js for raster data handling
  • Modern JavaScript for dynamic functionality
  • Responsive CSS for cross-device compatibility

Supported Formats

  • Input: GeoTIFF satellite imagery
  • Output: CSV with geographic coordinates and class labels

Use Cases

  • Creating training datasets for machine learning models
  • Land cover classification projects
  • Environmental monitoring studies
  • Urban planning and development analysis
  • Agricultural land use mapping
