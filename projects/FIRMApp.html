About this project

Forest Incident Reporting Module (FIRM) Android App is a multipurpose mobile application that has been developed in-house by the Tripura Forest Department. This mobile app has different modules like Wildlife location, Seizure, Patrolling, and Forest Fire. FIRM App has been developed using Java with API /backend web application in PHP. Several versions of the FIRM App have been released till now for the use of the officials of the Tripura Forest Department. (https://forest.tripura.gov.in/root/doc/2bf81f250b8a.pdf)

In order to overcome the difficulties mentioned above in monitoring and preparing a database on wildlife, a web GIS solution was developed in-house in the department. In a nutshell, it consists of an android mobile app and a web GIS portal for viewing data.

The web GIS application is hosted as part of the official website of the Tripura Forest Department (https://forest.tripura.gov.in/) and is accessible only for authorized users. There are four authorized roles/users categories: the Admin, FHQ Officer, SDFO/WLW, and Field Official. Each category has different duties/privileges and access to the system.

This project won the Silver Medal for digital governance products at the SKOCH Awards 2023 https://award.skoch.in/2023/05/09/tripura-forest-department/.
